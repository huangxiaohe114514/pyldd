# Windows Dynamic Island

一个用 PyQt6 开发的 Windows 版灵动岛软件，提供实时系统状态监测和流畅的动画效果。

## ✨ 功能特性

- 🕐 **实时时间显示** - 正常情况下显示当前时间（时:分:秒）
- 🖱️ **鼠标悬停展开** - 鼠标移到灵动岛上后平滑展开，显示 CPU 利用率
- 🌐 **网络状态监测** - 自动检测网络连接/断开，从灵动岛扩大显示通知
- 🔵 **蓝牙状态监测** - 自动检测蓝牙连接/断开，从灵动岛扩大显示通知
- 🎨 **流畅动画** - 60fps 平滑动画，仿 iPhone 灵动岛弹出方式
- 📊 **CPU 负载颜色指示** - 根据 CPU 负载显示不同颜色（绿色/黄色/红色）
- 🎯 **可拖动** - 支持鼠标拖动灵动岛到任意位置
- 🔒 **位置锁定** - 可锁定灵动岛位置，防止误触移动
- 🔄 **位置重置** - 一键重置灵动岛到屏幕中央

## 🚀 快速开始

### 环境要求

- Windows 10/11
- Python 3.8+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python dynamic_island.py
```

## 📖 使用说明

### 基本操作

- **查看时间**：灵动岛默认显示当前时间
- **查看 CPU 使用率**：将鼠标悬停在灵动岛上，它会展开并显示 CPU 利用率
- **拖动灵动岛**：按住鼠标左键拖动灵动岛到任意位置
- **锁定位置**：右键点击 → 选择"锁定灵动岛"
- **解除锁定**：右键点击 → 选择"解除锁定"
- **重置位置**：右键点击 → 选择"重置位置"
- **退出程序**：
  - 双击灵动岛即可退出
  - 或右键点击灵动岛，选择"退出"

### 状态指示

CPU 利用率颜色说明：
- 🟢 **绿色**：< 50% - 正常负载
- 🟡 **黄色**：50% - 80% - 中等负载
- 🔴 **红色**：> 80% - 高负载

### 通知动画

网络和蓝牙状态变化时，灵动岛会以 iPhone 灵动岛的方式弹出通知：
- 从胶囊形状平滑扩大到长方形
- 显示状态信息 3 秒后自动恢复
- 带有彩色边框指示状态类型

## 🎯 核心组件

### NetworkMonitor
网络状态监测器，定期检查网络连接状态，状态变化时发送信号。

### BluetoothMonitor
蓝牙状态监测器，通过 Windows 注册表和网络接口检查蓝牙状态。

### DynamicIsland
主灵动岛组件，包含时间显示、CPU 监测、拖动、锁定和通知动画功能。

## 📦 打包说明

如需自行打包：

```bash
pip install pyinstaller
pyinstaller --onefile --noconfirm --windowed --name DynamicIsland dynamic_island.py
```

打包后的文件位于 `dist/DynamicIsland.exe`
