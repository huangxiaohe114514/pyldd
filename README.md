# Windows Dynamic Island

一个用 PyQt6 开发的 Windows 版灵动岛软件，提供实时系统状态监测和流畅的动画效果。

## ✨ 功能特性

- 🕐 **实时时间显示** - 正常情况下显示当前时间（时:分:秒）
- 🖱️ **鼠标悬停展开** - 鼠标移到灵动岛上后平滑展开，显示 CPU 利用率
- 🌐 **网络状态监测** - 自动检测网络连接/断开，并弹出通知
- 🔵 **蓝牙状态监测** - 自动检测蓝牙连接/断开，并弹出通知
- 🎨 **流畅动画** - 60fps 平滑动画和淡入淡出效果
- 📊 **CPU 负载颜色指示** - 根据 CPU 负载显示不同颜色（绿色/黄色/红色）

## 🚀 快速开始

### 环境要求

- Windows 10/11
- Python 3.8+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python dynamic_island.py
```

## 📖 使用说明

### 基本操作

- **查看时间**：灵动岛默认显示当前时间
- **查看 CPU 使用率**：将鼠标悬停在灵动岛上，它会展开并显示 CPU 利用率
- **退出程序**：
  - 双击灵动岛即可退出
  - 或右键点击灵动岛，选择"退出"

### 状态指示

CPU 利用率颜色说明：
- 🟢 **绿色**：< 50% - 正常负载
- 🟡 **黄色**：50% - 80% - 中等负载
- 🔴 **红色**：> 80% - 高负载

## 🎯 核心组件

### NetworkMonitor
网络状态监测器，定期检查网络连接状态，状态变化时发送信号。

### BluetoothMonitor
蓝牙状态监测器，通过 Windows 注册表和网络接口检查蓝牙状态。

### NotificationWidget
通知窗口组件，显示网络和蓝牙状态变化的通知，带有淡入淡出动画。

### DynamicIsland
主灵动岛组件，包含时间显示、CPU 监测和展开/收起动画。
